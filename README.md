# 🦷 Dental Appointment Scheduler

A modern and responsive **Dental Appointment Scheduler** built with **React** and **Vite**. This application helps dental clinics efficiently manage patient appointments through an intuitive calendar interface.

---

---

## ✨ Features

- 📅 Interactive Monthly Calendar
- 👤 Add Patient Appointments
- 🦷 Doctor Assignment
- 🕒 Appointment Time Selection
- 📝 Treatment / Notes
- 💾 Local Storage Persistence
- 🔵 Appointment Indicator on Calendar
- 📊 Appointment Counter
- 🗑 Delete Appointments
- 📭 Beautiful Empty State
- ⏳ Loading Screen
- 📱 Fully Responsive Design
- ♿ Accessibility-Friendly UI
- 🛡️ Input Sanitization against XSS
- 📈 Simulated Analytics Logging

## 🚀 Tech Stack

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/react-calendar-widget.git
```

Go to the project folder

```bash
cd react-calendar-widget
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
